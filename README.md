# acme-reference-impl

A reference implementation project with 

- Java 8
- Jersey
- AngularJS
- Swagger
- Log4j2
- Hk2 DI
- Unit and Integration Tests
- Gradle
- Bower
- JPA + Hibernate

clone and run **gradlew appRun**

launch ***http://localhost:8080/acme-reference-impl/ui/swagger-ui/index.html***
